# 01-Rock-Paper-Scissors
This project is a Python-based implementation of the popular game "Rock, Paper, Scissors".  It provides the opportunity to play the game against a computer, which makes its moves based on random choice.

## Project Structure
```
rock-paper-scissors/
├── src/
│   ├── game.py          # Main game logic
├── README.md            # Project documentation
```

## Requirements
- Python 3.6 or higher : This project uses several Python 3.6+ features, including type hints and f-strings.
```python
pip install -r requirements.txt
```

## How to Run
1. Clone the repository to your local machine.
2. Navigate to the `src` directory.
3. Run the game using the command:
```bash
python game.py
```
4. Follow the on-screen prompts to play the game.
5. After each round, you will be asked if you want to play again or quit.
